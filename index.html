WOW vs wow

"WOW" is when you see a visualization and lean back and say "WOW!"

"wow" is when you see a visualization and lean foward and go "wow"

You want the latter

Don't get cute

  • Not using flash
  • Using Responsive Design

  • Showing our work on github
